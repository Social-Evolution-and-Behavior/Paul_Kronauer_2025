This directory contains the code and data used to generate Figure 1E in Paul et al. 2025.

The subdirectories Old and Young, contain the tracking data needed to run the code and generate the plot.

If you run the Jupyter Notebook on your system (Code_for_Figure_1E_Paul_et_al_2025.ipynb), be sure to update the file paths to the Old and Young subdirectories on your system accordingly.

The 'output_files' directory, contains the expected output plot of running the code.
